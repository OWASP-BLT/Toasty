Body: Creates a Probot-based GitHub App that responds to `/plan` slash commands in issue comments, generating structured implementation plans.

## Core Implementation

- **`app.js`** - Webhook handler listening for `issue_comment.created` events
  - Parses `/plan` command from comment body
  - Generates 6-step implementation plan from issue title/description
  - Posts plan as issue comment via GitHub API

```javascript
app.on("issue_comment.created", async (context) => {
  const comment = context.payload.comment.body;
  if (comment.trim().startsWith("/plan")) {
    const plan = generatePlan(issue.title, issue.body);
    await context.octokit.issues.createComment(context.issue({ body: plan }));
  }
});
```
